# Health Guardian AI

A simple, reliable AI system for early chronic disease prediction.

## Features
- **Multi-Disease Prediction**: Diabetes, Heart Disease, Kidney Disease.
- **Web Interface**: Clean, dark-mode UI using Streamlit.
- **Local AI**: No data leaves your machine.
- **Automated Setup**: One-click data download and training.

## Installation
1. Ensure you have Python installed.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Quick Start
Double-click **`run_app.bat`** to start the application.

## Developer Notes
- `src/app.py`: Main application logic.
- `src/train_model.py`: Script to retrain models.
- `download_data.py`: Script to fetch fresh datasets.
